Block Blast

Block Blast is an offline game. It is similar to Tetris. The game has two mini games. In the first mini game, the goal is to eliminate the blocks either vertically or horizontally to get a high score. The game will first give you three random blocks which you will put in the 8x8 grid. The second mini game has the same goal as the first but as a challenge, find out the other details by yourself. Enjoy playing!

5 reasons why this game is my favorite:

  • The game is very fun!
  • It is very addicting!
  • It's an offline game meaning I don't need an internet connection.
  • It makes me think critically because I have to calculate where I should put the blocks.
  • I like the thrill because I always want to beat my current record.
